# charts

## Installation
1. Install any non-built-in modules or packages<br>
```shell
pip install chart_studio
pip install pandas
pip install plotly
pip install Flask
```
2. Run server script<br>
```shell
cd servers/
python server.py
```
3. Open HTML via running Flask server
